The goal of Vache Taureau game is to know as quickly as possible a number chosen by the application. The Application can only choose numbers containing 4 digits and these digits are distinct two by two. At each guess the computer gives feedback as follows:

  • - If your proposed number contains a correct number but which is out of place, the computer displays: Vache(V).
  • - If your proposed number contains a correct number and which is in its place, the computer displays: Taureau (T).
